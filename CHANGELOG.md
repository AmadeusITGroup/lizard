## [0.1.0] - 2026-02-17

### Added

- Initial LIZARD fraud pattern visualizer
- Investigation dashboard with timeline, globe, map, pie, bar, scatter, and link graph panels
- AI-assisted data mapping with OpenAI integration
- Simple anomaly detection (Z-Score, EWMA, MAD)
- Advanced anomaly detection (Isolation Forest)
- Cloud integration with Azure Blob Storage, ADLS Gen2, and Databricks
- Background scheduler for cloud mode
- Data workbench with query builder
- Rules engine for fraud detection
- Docker deployment configuration
- CI/CD pipeline with GitHub Actions
- Comprehensive README documentation
